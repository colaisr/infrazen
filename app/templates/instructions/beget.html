Инструкция по подключению Beget

Пошаговое руководство для интеграции Beget с InfraZen

Требования

  • Активный аккаунт Beget
  • Доступ к панели управления Beget
  • Права администратора в InfraZen

1Получение учетных данных Beget

Имя пользователя

Ваше имя пользователя (логин) в системе Beget. Обычно это имя вашего аккаунта.

Пароль

Пароль от вашего аккаунта Beget.

Важно: InfraZen безопасно хранит ваши учетные данные в зашифрованном виде.

2Добавление подключения в InfraZen

  1. Перейдите в раздел "Подключения"
  2. Нажмите кнопку "+ Добавить подключение"
  3. Выберите провайдера "Beget" из выпадающего списка
  4. Заполните поля:
    • Название подключения: Любое удобное имя (например, "Beget Production")
    • Имя пользователя: Ваш логин Beget
    • Пароль: Ваш пароль Beget
  5. Нажмите "Тест подключения" для проверки
  6. Если тест успешен, нажмите "Сохранить"

3Первая синхронизация

После сохранения подключения InfraZen автоматически начнет синхронизацию:

  • VPS серверы (конфигурация, использование CPU/RAM/HD)
  • Виртуальный хостинг
  • S3 хранилища
  • MySQL базы данных
  • Стоимость всех ресурсов
Совет: Первая синхронизация может занять 1-2 минуты в зависимости от количества ресурсов.

Нужна помощь?

Если у вас возникли проблемы с подключением Beget, свяжитесь с нашей службой поддержки или ознакомьтесь с документацией Beget.
